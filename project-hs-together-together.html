FEATURED PLANNING AMSTERDAM 16 MAY 2026

Harry Styles — “Together, Together”

Fanproject for the Amsterdam show. Updates and participation will be handled through this site.

Join Project WhatsApp Community

Not an official project. Fan-led coordination only.

Status / Description

We’re currently still planning this project.

The idea is to create an animated lightshow that runs across participants’ phone screens. Right now we’re still figuring out the best way to implement it.

Since this is the opening show, we can only plan parts of it for now because we don’t know exactly how it will play out on the day. One approach we’re considering is using beat detection, which would trigger different color animations on phones, like a beat / music visualizer.

How to participate

  • A Google Drive will be created where all materials and info will be shared. (Coming soon.) Helpers will also be listed there.
  • If you want to promote the project (socials, groups, friends), that’s very welcome. We’re genuinely grateful for it.
