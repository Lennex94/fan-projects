Status / Description

We are currently still planning this project.

The idea is to create an animated lightshow that runs across participants' phone screens. Right now we are still figuring out the best way to implement it.

Since this is the opening show, we can only plan parts of it for now because we do not know exactly how it will play out on the day. One approach we are considering is using beat detection, which would trigger different color animations on phones, like a beat / music visualizer.

How to participate

  • Follow the social updates to get timing and instructions as the date gets closer.
  • Helpers and co-organizers wanted. Planning, design, coordination, anything helps. The more people, the better.
  • Once the concept is locked, we will create info sheets that need to be handed out in the stadium. If you can help with distribution, please reach out early.
  • A Google Drive will be created where all materials and info will be shared. (Coming soon.) Helpers will also be listed there.
  • If you want to promote the project (socials, groups, friends), that is very welcome. We are genuinely grateful for it.
